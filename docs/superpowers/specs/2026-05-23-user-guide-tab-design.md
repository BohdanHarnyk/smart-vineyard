### 3. Accordion Styling & Animation (CSS)
Add styles in the `<style>` block to support smooth height transitions and rotation of the chevron arrow icon:
```css
.accordion-content-wrapper {
    max-height: 0;
    overflow: hidden;
    transition: max-height 0.35s cubic-bezier(0.4, 0, 0.2, 1), opacity 0.3s ease;
    opacity: 0;
}
.accordion-item.active .accordion-content-wrapper {
    max-height: 1200px; /* Large enough to fit text */
    opacity: 1;
}
.accordion-chevron {
    transition: transform 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}
.accordion-item.active .accordion-chevron {
    transform: rotate(180deg);
}
```

### 4. Interactive Accordion Logic (JavaScript)
A global helper function to toggle active states:
```javascript
function toggleGuideAccordion(index) {
    const items = document.querySelectorAll('.guide-accordion-item');
    const clickedItem = items[index];
    const isActive = clickedItem.classList.contains('active');
    
    // Close all other items for clean accordion behavior
    items.forEach(item => item.classList.remove('active'));
    
    if (!isActive) {
        clickedItem.classList.add('active');
    }
}
```

## Guide Content Details (Ukrainian text draft)

### Accordion 1: 📅 Календар Фаз & Сезонні роботи
* **Заголовок**: `📅 Календар Фаз & Сезонні роботи`
* **Опис**: Як платформа допомагає адаптувати агротехніку під реальну погоду.
* **Текст**:
  * **Суть**: Традиційний календар за датами часто підводить через кліматичні зміни. Платформа автоматично прив'язує роботи до **фаз розвитку лози**, які обчислюються на основі температурного накопичення (САТ).
  * **Покрокова інструкція**:
    1. **Перегляд поточної фази**: У верхній частині вкладки "Календар Фаз" ви побачите рекомендовану поточну фазу (наприклад, "Перед цвітінням"), підсвічену на основі ваших температурних записів.
    2. **Перемикання фаз**: Ви можете вручну клікати по колу будь-якої фази на таймлайні, щоб переглянути майбутні роботи чи минулі записи.
    3. **Робота з чек-листами**: Кожна фаза містить дві вкладки: **«Захист лози»** (які бакові суміші пестицидів внести) та **«Живлення»** (які добрива дати під корінь чи по листу).
    4. **Фіксація виконання**: Поставте галочку біля виконаної роботи. З'явиться поле вибору дати — оберіть фактичний день проведення операції. Всі відмітки автоматично зберігаються в пам'яті браузера.

### Accordion 2: 📊 Смарт-Аналітика & Температурний моніторинг (САТ/СЕТ)
* **Заголовок**: `📊 Смарт-Аналітика & Температурний моніторинг (САТ/СЕТ)`
* **Опис**: Розрахунок теплових ресурсів сезону та захист від приморозків і хвороб.
* **Текст**:
  * **Суть**: Лоза розвивається залежно від накопиченого тепла. Платформа підтримує два режими розрахунку:
    * **САТ (Сума Активних Температур)**: Сума середньодобових температур повітря, які є вищими або рівними +10°C. *Приклад: якщо за добу середня температура була +22°C, внесок у САТ становить 22°C.*
    * **СЕТ (Сума Ефективних Температур / GDD)**: Сума температур за вирахуванням біологічного нуля лози (+10°C). *Приклад: середня t = +22°C дає внесок у СЕТ = 12°C (22 - 10).*
  * **Покрокова інструкція**:
    1. **Внесення температури**: У вкладці "Аналітика" виберіть дату в календарі, введіть середньодобову температуру (наприклад, `23`) у поле введення та натисніть `➕ Додати запис`.
    2. **Перегляд графіка**: Інтерактивний графік показує криву накопичення тепла за сезон. Ви можете перемикати режими САТ/СЕТ за допомогою кнопок-перемикачів: графік перерахує дані автоматично.
    3. **Детектор заморозків**: Платформа моніторить прогноз приморозків (якщо погода синхронізована по API). При прогнозованій температурі нічної t <= 2.0°C з'явиться попереджувальна картка ризику.
    4. **Прогноз хвороб**: Детектор хвороб обчислює ризик спалаху мілдью за правилом **6-6-26**: якщо випало понад 6 мм опадів за температури повітря від 6°C до 26°C, з'явиться критичне попередження про інкубаційний період грибка.

### Accordion 3: 🍇 Енциклопедія Сортів & Розсадник (Школка)
* **Заголовок**: `🍇 Енциклопедія Сортів & Розсадник (Школка)`
* **Опис**: Ведення реєстру насаджень виноградника та управління саджанцями.
* **Текст**:
  * **Суть**: Інструмент для ведення точного обліку кількості дорослих кущів та молодих саджанців на етапі вкорінення.
  * **Покрокова інструкція**:
    1. **Перегляд колекції**: У під-вкладці **«Моя колекція»** ви знайдете 122 базових сорти винограду. Клік на сорт відкриває його детальний агропаспорт.
    2. **Додавання сорту**: Натисніть `➕ Додати сорт`. У модальному вікні почніть вводити назву. Завдяки **автозаповненню**, якщо ви введете стандартне ім'я (наприклад, «Юпітер»), система сама підтягне термін дозрівання, колір, смак та опис догляду. Вкажіть початкову кількість кущів та збережіть сорт.
    3. **Облік кущів**: На картці паспорта сорту є лічильник кількості кущів. Використовуйте кнопки `+` та `-` для швидкого оновлення кількості (наприклад, якщо досадили 2 кущі або видалили старий). Кількість кущів з'явиться у вигляді бейджа на плитці сорту в списку.
    4. **Робота з Розсадником (Школкою)**: Перейдіть на під-вкладку **«Розсадник / Школка»**. Сюди можна додавати молоді саджанці чи чубуки на вкорінення. Коли вони виростуть і будуть готові до висадки в поле, натисніть кнопку **🌱 Висадити**, ввести кількість (наприклад, `3`). Система сама спише їх зі школки і додасть відповідну кількість кущів у вашу «Мою колекцію», оновивши всі баланси.

### Accordion 4: 🧮 Калькулятори Виноградаря
* **Заголовок**: `🧮 Калькулятори Виноградаря (NPK, Оприскування, Фертигація)`
* **Опис**: Математичні розрахунки дозувань добрив та приготування робочих розчинів.
* **Текст**:
  * **Калькулятор NPK (Сухі добрива)**:
    * *Завдання*: Розрахувати вагу комерційних добрив для внесення під корінь навесні чи восени.
    * *Крок*: Введіть цільову потребу в чистих елементах живлення на кущ (Азот N, Фосфор P, Калій K) та оберіть комерційне комплексне добриво (наприклад, YaraMila Complex NPK 12-11-18). Програма розрахує вагу комплексного добрива і точно обчислить вагу простих добрив (селітра, монофосфат), які потрібно додати для компенсації дефіциту.
  * **Захист & Оприскування (Бакова суміш)**:
    * *Завдання*: Підготувати робочий розчин пестицидів без хімічного опіку лози та нейтралізувати жорсткість води.
    * *Крок*: Вкажіть об'єм обприскувача (наприклад, 10 літтів) та введіть препарати в таблицю (вказавши дозування на 1 літр води). Калькулятор видасть рецепт із загальною вагою кожної речовини та порахує дозування лимонної кислоти для підкислення води (до pH 5.5-6.0), щоб запобігти розпаду пестицидів у лужній воді.
  * **Калькулятор Фертигації (Крапельне живлення)**:
    * *Завдання*: Внесення розчинених добрив через крапельну систему з контролем засолення (EC).
    * *Крок*: Вкажіть кількість кущів, водовилив однієї крапельниці та бажану дозу добрив. Калькулятор розрахує час поливу та концентрацію солей (EC). **Важливо**: якщо розрахований індекс EC перевищує 2.0 mS/cm, система покаже червоне попередження — розчин є занадто концентрованим, що може призвести до опіку кореневої системи. Також розраховується вага добрив для маточного розчину при використанні інжектора Вентурі.

### Accordion 5: 🧪 Лабораторія & ГК (Гіберелінова кислота)
* **Заголовок**: `🧪 Лабораторія & ГК (Гіберелінова кислота)`
* **Опис**: Розрахунок концентрацій стимуляторів росту (GA3) для обробки суцвіть та ягід.
* **Текст**:
  * **Суть**: Обробка гібереліном допомагає розтягнути гроно, зменшити щільність ягід (профілактика гнилей) та збільшити розмір ягід (особливо у кишмишів).
  * **Покрокова інструкція**:
    1. **Вхідні дані**: Введіть вміст діючої речовини у вашому препараті у % (наприклад, `20` для порошку або `100` для чистої кислоти).
    2. **Цільові параметри**: Вкажіть потрібний об'єм розчину (наприклад, `1` літр) та концентрацію в ppm (міліграм на літр). *Типові концентрації: 10-20 ppm для розтягування суцвіть до цвітіння; 30-40 ppm для збільшення ягід безнасінних сортів після цвітіння.*
    3. **Розрахунок**: Калькулятор видасть необхідну точну вагу порошку препарату.
    4. **Спиртовий розчин**: Гіберелін не розчиняється безпосередньо у воді. Спочатку розчиніть розраховану наважку у невеликій кількості (10-20 мл) спирту 96%, а вже потім вилийте отриманий розчин у воду!

### Accordion 6: 🔬 AI Асистент & Розумний підбір сортів
* **Заголовок**: `🔬 AI Асистент & Розумний підбір сортів`
* **Опис**: Інтелектуальний консультант з вирощування та перевірка кліматичної сумісності.
* **Текст**:
  * **Суть**: Чат-бот на базі штучного інтелекту Gemini, адаптований під професійні питання виноградарства.
  * **Покрокова інструкція**:
    1. **Активація ключа**: У шапці вкладки введіть свій унікальний Gemini API ключ та збережіть його. Платформа зв'яжеться з серверами ШІ напряму з вашого комп'ютера, зберігаючи повну конфіденційність ключа.
    2. **Детектор сумісності**: Введіть назву сорту винограду та температурний показник САТ вашого регіону (наприклад, `2600`). AI миттєво розрахує групу стиглості сорту та видасть висновок: чи дозріє він повністю у вашій зоні, чи є ризик недобору цукру.
    3. **Голосовий чат**: Клікніть на значок мікрофона `🎙️` поруч із полем введення в Agro-Chat. Промовте запит українською мовою (наприклад, *"Як боротися з антракнозом під час дощів?"*). Система розпізнає ваш голос, запише текст і після натискання кнопки відправки надішле запит.
    4. **Зберігання історії**: Усі діалоги автоматично кешуються в пам'яті вашого браузера. Ви можете вільно перезавантажувати сторінку — історія листування не зникне, поки ви самі не очистите її кнопкою `Очистити діалог`.
    5. **Таблиця порівняння**: Якщо ви запитуєте AI про нові сорти, ви можете зберегти їх у "Кандидати на посадку". Вони з'явиться на бічній панелі чату, де можна порівнювати їх за термінами дозрівання перед покупкою саджанців.
