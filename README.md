# 🍇 Розумний Виноградник | Вінниччина Professional

**Smart Vineyard** — це сучасний веб-додаток (Single Page Application) для точного виноградарства, адаптований під кліматичні умови Вінницької області та України. Додаток розроблено для оптимізації процесів догляду за лозою, розрахунку дозування добрив, моніторингу температурних показників (активних температур) та захисту рослин від хвороб.

---

## 🌟 Основні можливості та розділи

### 1. 📅 Календар вегетаційних фаз
* **Інтерактивний чек-лист**: Покрокові завдання для 7 ключових фаз розвитку винограду (від набухання бруньок до підготовки до зими).
* **Відстеження прогресу**: Динамічне відображення відсотка виконання робіт для кожної фази.
* **Збереження стану**: Всі позначені пункти автоматично зберігаються в `localStorage` вашого браузера та не зникають при перезавантаженні сторінки.

### 2. 📝 Планувальник робіт та Журнал операцій
* **Персональні завдання**: Можливість додавати власні операції на винограднику з встановленням дедлайнів.
* **Автоматична фіксація дати**: При відмітці завдання як виконаного, воно автоматично переноситься до "Архіву виконаних робіт" із фіксацією поточної дати.
* **Ретроспективне редагування**: Можливість вручну змінити дату фактичного проведення операції безпосередньо в інтерфейсі журналу.

### 3. 📈 Смарт-Аналітика SAT (Сума Активних Температур)
* **Розрахунок SAT**: Формула розраховує ефективний тепловий внесок кожного дня за принципом:
  $$\text{Внесок} = T_{\text{середня}} - 10^\circ\text{C} \quad (\text{для днів з } T_{\text{середня}} > 10^\circ\text{C})$$
* **Графік накопичення**: Інтерактивна візуалізація кривої SAT за допомогою Chart.js.
* **Агрономічні орієнтири**: Шкала прогресу відносно ключових фаз (розпускання бруньок — 200°C, цвітіння — 800°C, розм'якшення ягід — 1000°C, повна зрілість — 2500°C).
* **Керування даними**: Можливість додавати записи за конкретні дати, видаляти помилкові записи або повністю скидати сезонний лог.

### 4. 🌦️ Прогнозування ризику хвороб
* **Моделювання ризику**: Інтерактивні повзунки для вибору поточної температури (°C), опадів (мм) та відносної вологості (%).
* **Алгоритм виявлення**:
  * **Мілдью**: Високий ризик за класичним правилом «6-6-26» (опади > 6 мм, температура між 6°C та 26°C).
  * **Оїдіум**: Високий ризик при температурі 15°C - 25°C та вологості > 70%.
* **Рекомендації**: Миттєві попередження та список дієвих фунгіцидів (Акробат, Танос, Луна Експіріенс, препарати сірки тощо).

### 5. 📚 Енциклопедія сортів (122 сорти)
* **Агрономічні паспорти**: Детальний опис для 122 сортів винограду (вага грона, період дозрівання, стійкість до хвороб, смак, сила росту та сумісність з гібереліном).
* **Фільтрація та Сортування**: Швидка фільтрація за кольором ягід (Білі, Рожеві, Чорні) та сортування за алфавітом, тривалістю дозрівання чи середньою вагою грона.
* **Підтримка української локалізації**: Повна сумісність інтерфейсу з українськими літерами (`і`, `ї`, `є`, `ґ`).

### 6. 🧮 Дворежимний калькулятор живлення
* **Простий режим**: Розрахунок загальної ваги добрива на основі кількості кущів та фізичної норми внесення на один кущ.
* **Режим за діючою речовиною**: Допомагає розрахувати масу комерційного добрива, виходячи з необхідної кількості чистого елемента живлення (N, P, K тощо) та його відсоткового вмісту в препараті.
* **Популярні пресети**: Швидкий вибір популярних добрив (YaraMila Complex, Карбамід, Аміачна селітра, Монокалійфосфат, Сульфат калію, Кальцієва селітра).

### 7. 🧪 Лабораторія Гібереліну (ГК)
* **Калькулятор розчинів**: Розраховує точну масу препарату в залежності від об'єму води, цільової концентрації в PPM (мг/л) та чистоти форми (90% порошок, 20% таблетка, 4% рідкий концентрат).
* **Інструкція приготування**: Нагадування про необхідність первинного розчинення ГК в спирті (96% етанол), оскільки речовина практично не розчиняється у воді безпосередньо.

## 🚀 Локальний запуск

Оскільки проект є статичним односторінковим додатком (SPA) на базі HTML, CSS та JavaScript:
1. Завантажте або склонуйте цей репозиторій.
2. Відкрийте файл `index.html` безпосередньо у будь-кому веб-браузері.
3. *Альтернативно*: запустіть локальний сервер за допомогою розширення Live Server у VS Code або команди:
   ```bash
   # За допомогою Python
   python -m http.server 8000
   ```
   Після чого відкрийте в браузері `http://localhost:8000`.

## ☁️ Деплой на Vercel

Проект повністю готовий до миттєвого безкоштовного деплою на платформу **Vercel**:

1. Зайдіть на [Vercel](https://vercel.com/) та авторизуйтесь через свій GitHub-акаунт.
2. Натисніть кнопку **Add New** -> **Project**.
3. Виберіть репозиторій `smart-vineyard` зі списку підключених репозиторіїв GitHub.
4. Натисніть кнопку **Deploy** (інші налаштування за замовчуванням підходять ідеально, оскільки проект є чистим HTML/JS додатком).
5. За декілька секунд ваш додаток буде опубліковано на публічному домені `*.vercel.app`.

## 🛠️ Технологічний стек
* **Розмітка**: HTML5 (семантична структура)
* **Стилі**: Tailwind CSS (через CDN для швидкої інтеграції)
* **Графіки**: Chart.js (для візуалізації накопичення SAT)
* **Логіка**: Чистий JavaScript (ES6+), збереження стану в `localStorage`
