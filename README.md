# 🍇 Розумний Виноградник | Smart Vineyard • Вінниччина Professional

**Smart Vineyard** — це передова цифрова агрономічна система та інтерактивна платформа для точного виноградарства, спеціально оптимізована під кліматичні умови України (зокрема Вінницької області). Проект реалізований як високопродуктивний Single Page Application (SPA), що працює повністю на стороні клієнта, забезпечуючи високу швидкість роботи, конфіденційність та автономність даних.

## 🌟 Ключові можливості та архітектурні модулі

### 1. 📅 Календар вегетаційних фаз
* **Інтерактивний чек-лист**: Система розбита на 7 ключових фаз розвитку виноградного куща (від набухання бруньок до осінньої обрізки та укриття на зиму).
* **Контроль операцій**: Кожна фаза містить детальні технологічні карти робіт (зелені операції, профілактичні обробки, підживлення).
* **Моніторинг виконання**: Графічні індикатори відображають прогрес виконання завдань для кожної фази окремо.
* **Синхронізація стану**: Будь-які позначені пункти миттєво зберігаються у `localStorage`, гарантуючи збереження даних при перезавантаженні сторінки чи втраті зв'язку.

### 📝 deAgro Smart Diary (Розумний щоденник робіт)
* **Мультимедійне логування**: Створення записів трьох категорій: `✍️ Робота` (загальна агротехніка), `🧪 Препарат` (ЗЗР, підживлення з автовибором з бази) та `⚠️ Загроза` (хвороби, кліщі чи заморозки з вказівкою рівня небезпеки).
* **Голосове введення (Speech-to-Text)**: Запис нотаток голосом безпосередньо з інтерфейсу за допомогою вбудованого українського розпізнавання мови.
* **ШІ-парсинг Gemini**: Магічна кнопка `✨` розпізнає наміри в надиктованому тексті, самостійно перемикає типи операцій, підбирає препарати з бази та заповнює дозування.
* **Meteo-Logging**: При перенесенні робіт до Журналу система автоматично фіксує поточні метеодані (t°, вологість, опади) з Open-Meteo API.
* **Ретроспективне редагування**: Користувач може вручну змінити дату фактичного проведення операції безпосередньо в інтерфейсі журналу для точного ведення агрономічної історії.

### 📈 deAgro Адаптивна Аналітика & ШІ-Аудитор
* **Сума Активних Температур (САТ)**: Накопичення тепла із середньодобовою температурою $\ge 10^\circ\text{C}$ за формулою:
  $$\text{САТ} = \sum (T_{\text{середня}} - 10^\circ\text{C}) \quad \text{при } T_{\text{середня}} > 10^\circ\text{C}$$
* **Сума Ефективних Температур (СЕТ / GDD)**: Альтернативний режим розрахунку теплових одиниць для відстеження точних фаз цвітіння та дозрівання ягід.
* **Аналітичний Дашборд**: Візуальне представлення агрохімічного балансу проведених робіт та детального кліматичного архіву сезону (кількість дощових днів, середня температура обробок).
* **🧠 Адаптивний ШІ-Аудитор**: Кнопка генерації ШІ-стратегії зчитує ваші сорти, накопичений САТ та всю історію журналу з метеометками для побудови експертного звіту: оцінки стану вегетації, виявлення агротехнічних помилок та планування наступних фаз.
* **Адаптивні попередження**: Автоматичний моніторинг опадів та азотного навантаження (NPK баланс) для миттєвого виведення попереджень про ризики мілдью або жирування лози.
* **Інтерактивний графік**: Візуалізація накопичувальної кривої сезону за допомогою бібліотеки `Chart.js` із нанесенням агрономічних маркерів (Брунька: 200°C, Цвітіння: 800°C, Розм'якшення ягоди: 1000°C).
* **Автоматична синхронізація з Open-Meteo API**: Отримання точних метеоданих за геокоординатами користувача (широта/довгота) та автоматичний прорахунок накопиченого САТ без необхідності ручного введення.
* **Ручне керування САТ**: Можливість додавати або видаляти записи за будь-яку дату для ручного калібрування логів.

### 🛡️ Калькулятор ризиків (Хвороби та Заморозки)
* **Прогнозування мілдью**: Аналіз ризику зараження за класичним правилом «6-6-26» (опади > 6 мм за температури в межах 6–26°C).
* **Прогнозування оїдіуму**: Виявлення сприятливих умов розвитку хвороби при помірній температурі (15–25°C) та відносній вологості повітря понад 70%.
* **Попередження приморозків**: Отримання 14-денного прогнозу мінімальних температур через API та автоматичний вивід попереджень, якщо температура падає нижче +2°C.
* **Екстрений протокол**: Покрокові дії за 20 годин та 3 години до заморозку (застосування антистресантів типу Мегафол, дощування, димлення).

### 🍇 Бібліотека сортів (122 сорти)
* **Агропаспорти**: Детальна база даних на 122 сорти винограду, що культивуються в Україні. Містить інформацію про колір ягоди, вагу грона, термін дозрівання (в днях), кісточковість (кишмиш чи столовий), смакові якості та особливості догляду.
* **Керування кущами**: Можливість вказати кількість кущів кожного сорту на вашому винограднику.
* **Сортування та пошук**: Миттєва фільтрація за кольором ягід, пошук за назвою та швидке сортування.
* **Розсадник / Школка**: Окремий підрозділ для обліку саджанців та чубуків у процесі вкорінення з можливістю фіксації кількості та джерела походження.

### 🗄️ База препаратів (Full CRUD)
* **Універсальний реєстр**: База засобів захисту рослин (фунгіциди, інсектициди, акарициди), стимуляторів та добрив.
* **Гнучке додавання та редагування**: Повне керування картками препаратів: назва, хімічна група, діючі речовини, сумісність у бакових сумішах, рН-рекомендації та терміни очікування до збору врожаю.
* **Захист системних даних**: 19 базових препаратів автоматично імпортуються до системи. При їх редагуванні зміни зберігаються локально, а при видаленні користувачем спрацьовує логіка «Reset to Default», що повертає базовий заводський опис препарату.
* **Інтеграція з AI**: Функція автоматичного заповнення картки препарату за допомогою штучного інтелекту (Gemini API) за однією лише назвою.

### 🧮 Розрахунковий центр (Калькулятори)
* **Калькулятор сухих добрив NPK**: 
  * *Простий режим*: розрахунок фізичної маси добрива на кущ.
  * *Режим за діючою речовиною*: розрахунок необхідної маси комерційного добрива для внесення точної дози чистого азоту (N), фосфору (P) чи калію (K) з пресетами та автосинхронізацією з Базою препаратів.
* **Калькулятор бакових сумішей**: Розрахунок концентрацій ЗЗР на будь-який об'єм обприскувача з контролем сумісності та норми виливу на сотку.
* **Лабораторія Гібереліну (ГК)**: Розрахунок точної маси кристалічного порошку або таблеток ГК для отримання цільової концентрації розчину в PPM (мг/л). Містить інструкцію щодо первинного розчинення у 96% спирті.

### 🔬 AI Помічник (Agro-Chat)
* **Контекстний чат-бот**: Інтелектуальний агроном, підключений через API, що відповідає на складні питання щодо бакових сумішей, лікування рідкісних хвороб та формує індивідуальні схеми обробок.
* **Локальне кешування**: Збереження повної історії діалогів безпосередньо у вашому браузері.

## 🛠️ Технологічний стек

* **Frontend Core**: Чистий HTML5 та сучасний Vanilla JavaScript (ES6+).
* **Стилізація**: Tailwind CSS (чуйний grid, сучасна палітра кольорів, адаптивність під мобільні пристрої).
* **Графіки**: `Chart.js` для візуалізації термо-трендів.
* **Бази даних**: `LocalStorage` для безсерверного збереження користувацьких налаштувань, бази препаратів та журналів робіт.
* **Інтеграції**: 
  * Open-Meteo Weather & Archive API (синхронізація погоди та розрахунок кліматичних показників).
  * Google Gemini API (інтелектуальне автозаповнення сортів/препаратів та інтерактивні консультації).

## 🚀 Локальний запуск та розгортання

### Локально:
Проект є повністю статичним, тому не потребує встановлення важких залежностей типу Node.js чи Docker для запуску:
1. Завантажте або склонуйте цей репозиторій:
   ```bash
   git clone https://github.com/BohdanHarnyk/smart-vineyard.git
   ```
2. Відкрийте файл `index.html` (основна версія з повною аналітикою та AI) або `index_clean.html` (полегшена версія) безпосередньо у будь-якому сучасному веб-браузері.
3. *Альтернативно* (рекомендовано для коректної роботи деяких API запитів): запустіть локальний сервер:
   ```bash
   python3 -m http.server 8000
   ```
   та відкрийте в браузері `http://localhost:8000`.

### Деплой на GitHub Pages:
Проект налаштовано на деплой через GitHub Pages з гілки `main`, корінь `/`. При пуші в репозиторій зміни миттєво стають доступними на хостингу.
